# Fokus - Temporizador Pomodoro

O **Fokus** é um projeto desenvolvido para praticar a manipulação do DOM, utilizando HTML, CSS e JavaScript. O aplicativo segue a metodologia **Pomodoro** para ajudar a melhorar a produtividade, dividindo o tempo em intervalos de trabalho e descanso.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- **Temporizador Pomodoro**: Configurações de tempo para trabalhar (25 minutos) e para descanso curto (5 minutos) e descanso longo (15 minutos).
- **Iniciar/Parar**: O usuário pode iniciar e parar o temporizador a qualquer momento.
- **Contagem regressiva**: O tempo é exibido dinamicamente na interface.
- **Notificação de Fim de descanso**: O aplicativo avisa quando o tempo de descanso se esgota.
